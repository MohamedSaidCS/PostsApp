# Objects Task - Social Media App

# Technologies used
1) React
2) TypeScript
3) React Router Dom
4) Tailwind CSS
5) Axios


# Instructions

1) Clone the project
```
git clone https://github.com/MohamedSaidCS/PostsApp.git
```
2) Go to project directory
```
cd PostsApp
```
3) Install dependencies using npm
```
npm install
```
4) Run the application
```
npm run dev
```
